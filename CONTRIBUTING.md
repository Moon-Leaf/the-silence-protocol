# Contributing to The Silence Protocol

Thank you for your interest! This project is still in its early days, but every contribution matters.

## How to Contribute

1. **Read the basics**  
   Familiarize yourself with the [README](README.md) and the [FAQ](docs/FAQ.md).

2. **Report a problem or idea**  
   Open an [Issue](../../issues) and describe clearly what you have in mind — bug, feature, feedback, anything.

3. **Make changes**  
   - Fork this repository  
   - Create a new branch (`git checkout -b feature/your-contribution`)  
   - Commit your changes with a clear message  
   - Push to your fork and open a Pull Request  

4. **Respect the community**  
   Please follow our [Code of Conduct](CODE_OF_CONDUCT.md).

We review Pull Requests within 5 working days — no notification anxiety here.
