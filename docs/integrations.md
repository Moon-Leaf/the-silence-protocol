# Integrations – Supported Tools & Platforms

This is a list of integrations that are currently available, in development, or planned. The Silence Protocol is **not a product**, but a protocol; the integrations below are implementations of the technical specification.

## Integration Status

| Platform | Current Support | Status |
|---|---|---|
| **Slack** | Status sync, message queuing bot | ✅ Available |
| **Apple/macOS Focus Modes** | DND profile mapping | ✅ Available |
| **Android DND** | Schedule config templates | ✅ Available |
| **Microsoft Teams** | Manual status mapping guide | 📄 Docs only |
| **Notion / Linear** | Availability widget templates | 📄 Docs only |
| **Calendar (Google, Outlook)** | Focus block to calendar sync | 🔜 Planned |
| **Native mobile app** | Full protocol management | 🔜 Planned (by community) |

## How to Contribute
See [CONTRIBUTING.md](../CONTRIBUTING.md) to add new integrations.  
We welcome scripts, plugins, or configurations for any platform.

## Note
All integrations must follow the [Technical Spec](technical-spec.md) and contain **no dark patterns**.
