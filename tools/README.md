# Tools – Scripts, Configs & Bots

This folder contains various helpers to implement The Silence Protocol. These tools are **not a product**; they are reference implementations of the technical specification.

## Available Tools

| Tool | Description | Status |
|---|---|---|
| `focus-timer/` | CLI Pomodoro-style focus timer | 🚧 In development |
| `notification-configs/` | iOS/Android notification profiles | ✅ Available |
| `slack-etiquette-bot/` | Slack bot to honor Response Windows | ✅ Available |

## Usage
Each subfolder has its own README. Browse the one you need.

## Contributing
We welcome new tools! Please read [CONTRIBUTING.md](../CONTRIBUTING.md) before submitting a PR.

## Principles
All tools must respect:
- **Silence by Default** – no unsolicited interruptions.
- **No Dark Patterns** – no psychological tricks.
- **Transparent Unavailability** – status is always clear.
